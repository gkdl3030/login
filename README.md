# 회원관리 프로젝트 

1.memDB를 만든다.<br>
2.user_tbl을 만든다.

# 순서
1. 회원가입 누른다.<br>
2. 회원가입 양식에 맞춰 입력한다.
3. 회원가입을 하면 등록이 됐다고 뜬다.
4. 로그인창이 뜬다.
5. 로그인이 완료됐다고 뜬다.
6. 로그아웃
